# UniAid Student Management System

A comprehensive **Student Management System** designed to streamline student admissions, recruitment, application processing, and educational consultancy operations.

## 📌 Project Structure

This repository contains multiple branches, each dedicated to a specific platform or service within the UniAid ecosystem.

| Branch            | Company                                                                 |
| ----------------- | --------------------------------------------------------------------------- |
| `main`            | **UniAid**                     |
| `raya`            | **Recruitment**  |
| `iapply`          | **iApply**           |
| `watney-dubai`    | **Watney Dubai**   |

---

## 🚀 Features

* Student Management
* Application Tracking
* Recruitment Management
* Agent & Consultant Management
* User Role & Permission Management
* Dashboard & Analytics
* Responsive Admin Panel
* Secure Authentication & Authorization
